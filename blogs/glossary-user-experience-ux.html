User Experience (UX) Glossary

User Experience

The quality of an interaction with a product, system, or service. It encompasses how a person feels about using a product, system, or service, focusing on efficiency, effectiveness, and satisfaction.

User Interface (UI)

The aspect of a product that a user interacts with, including the design, layout, and content. It focuses on how the product looks and feels to the user.

Usability

The ease with which a user can use a product to achieve specific goals, with the user interface and the user's skill level being the primary factors.

User Testing

The process of observing users as they interact with a product or system to gather feedback and insights about the user experience.
