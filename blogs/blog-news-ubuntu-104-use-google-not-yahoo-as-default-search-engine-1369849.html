Switching Default Search Engine to Google in Ubuntu 10.4

In Ubuntu 10.4, users can change the default search engine from Yahoo to Google. This can be done by modifying the search preferences.

Steps to Switch Search Engine

1. Open the Unity Launcher and search for "Software & Updates".

2. In the "Software & Updates" window, go to the "Ubuntu Software" tab.

3. Look for the "Search" section and select "Google" from the dropdown menu.

4. Click "Apply" and then "Close" to save the changes.
